# 🌍 AI Travel Grouping & Planning System

This project uses Machine Learning (K-Means clustering) to group travelers based on preferences like budget, interest, and travel duration. It also recommends destinations and integrates real-time weather data using OpenWeather API.

## 🚀 Features
- User grouping using AI
- Travel destination recommendation
- Weather API integration
- Interactive dashboard (Streamlit)
- Login system

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- OpenWeather API

## 📊 Output
Displays grouped users, recommended destinations, and real-time weather.
